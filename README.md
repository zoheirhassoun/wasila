# وسيلة (Wasila)

PWA على Netlify: متاجر، توصيل ركاب، توصيل مطاعم، حجوزات طيران، وبوابة شركاء.

## التشغيل المحلي

```bash
npm install
netlify dev
```

يفتح الواجهة والدوال معاً. للتطوير دون دوال:

```bash
npm run dev
```

## البناء والنشر

```bash
npm run build
```

انشر عبر Netlify: ربط المستودع (Git) أو استخدام Netlify CLI. للخطوات التفصيلية: [docs/النشر-على-Netlify.md](docs/النشر-على-Netlify.md).

## التثبيت على الجوال (أندرويد / iOS)

التطبيق PWA: بعد النشر، يمكن تثبيته من المتصفح كتطبيق (إضافة إلى الشاشة الرئيسية). لتفاصيل التثبيت ولبناء نسخة للمتاجر (Capacitor) راجع [docs/التثبيت-على-الجوال.md](docs/التثبيت-على-الجوال.md).

## المتغيرات البيئية (اختياري)

- `PARTNER_API_KEY`: مفتاح API للشركاء (لحماية `/api/partner/*`).

## المسارات

- `/` الرئيسية
- `/stores` متاجر — `/stores/:id` تفاصيل ومنتجات
- `/rides` توصيل ركاب
- `/restaurants` مطاعم — `/restaurants/:id` قائمة طعام
- `/flights` حجوزات طيران
- `/more` المزيد
- `/partner` انضم كشريك
- `/partner/dashboard` لوحة الشريك
- `/partner/api-docs` توثيق API

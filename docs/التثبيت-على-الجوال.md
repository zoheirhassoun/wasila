# تثبيت وسيلة على الجوال (أندرويد و iOS)

## الطريقة الأولى: التثبيت كتطبيق PWA (بدون متجر)

التطبيق يعمل كـ PWA. بعد نشره على رابط (مثلاً Netlify)، يمكن تثبيته على الجوال مباشرة.

### أندرويد (Chrome)
1. افتح الرابط في متصفح **Chrome**.
2. اضغط قائمة المتصفح (النقاط الثلاث) أو رسالة "تثبيت التطبيق" إن ظهرت.
3. اختر **"إضافة إلى الشاشة الرئيسية"** أو **"تثبيت التطبيق"**.
4. اضغط **تثبيت** في النافذة الظاهرة.
5. سيظهر أيقونة "وسيلة" في الشاشة الرئيسية وتعمل كتطبيق.

### iOS (Safari)
1. افتح الرابط في متصفح **Safari** (لا يستخدم Chrome للتثبيت).
2. اضغط زر **المشاركة** (المربع مع السهم للأعلى).
3. اختر **"إضافة إلى الشاشة الرئيسية"**.
4. عدّل الاسم إن أردت واضغط **إضافة**.
5. سيظهر أيقونة "وسيلة" على الشاشة الرئيسية وتعمل كتطبيق.

## الطريقة الثانية: بناء نسخة للنشر على المتاجر (اختياري)

إذا أردت لاحقاً نشر التطبيق على **Google Play** أو **App Store** كملف تحميل (.apk / .aab أو .ipa)، استخدم **Capacitor** لتحويل المشروع إلى تطبيق أندرويد و iOS.

### المتطلبات
- Node.js
- أندرويد: [Android Studio](https://developer.android.com/studio) (للبناء والتوقيع)
- iOS: جهاز Mac مع [Xcode](https://developer.apple.com/xcode/) (للبناء ورفع App Store)

### خطوات سريعة

```bash
# 1. بناء المشروع
npm run build

# 2. إضافة Capacitor
npm install @capacitor/core @capacitor/cli
npx cap init "وسيلة" "com.wasila.app"

# 3. إضافة منصات أندرويد و iOS
npm install @capacitor/android @capacitor/ios
npx cap add android
npx cap add ios

# 4. نسخ ملفات البناء إلى المشروع الأصلي
npx cap sync

# 5. فتح المشروع في Android Studio أو Xcode
npx cap open android   # لأندرويد
npx cap open ios        # لـ iOS (على Mac فقط)
```

ثم من **Android Studio**: Build → Build Bundle(s) / APK(s) لإنشاء ملف التثبيت.  
ومن **Xcode**: اختيار جهاز أو محاكي ثم Archive لإنشاء نسخة للنشر أو التوزيع.

### ملاحظات
- تأكد أن عنوان الموقع (origin) في الإنتاج يستخدم **https** حتى تعمل الـ API والسلة بشكل صحيح.
- لـ **App Store** تحتاج حساب مطوّر Apple (مدفوع) ومراجعة التطبيق.
- لـ **Google Play** تحتاج حساب مطوّر Google (دفعة واحدة) ورفع التطبيق.
